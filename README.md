#  Football Match Performance Analysis

## Project Overview
The "Football Match Performance Analysis" project focuses on analyzing football match data to gain meaningful insights into team and player performance. The dataset is dynamically generated using a Python script and analyzed through interactive visualizations. The project helps users understand match outcomes, goal statistics, player performance, and overall team performance using key performance metrics.

---

##  Objectives
- Analyze football match performance.
- Compare goals scored and goals conceded.
- Track wins, losses, and draws.
- Evaluate player performance.
- Present insights through interactive visualizations.
- Demonstrate dynamic data analysis using Python and Power BI.

---

##  Tools & Technologies
- Python
- Power BI
- DAX

- CSV Dataset

---

##  Dataset
The dataset is generated using Python and contains the following columns:

- Match_ID
- Goals_Scored
- Goals_Conceded
- Result
- Player
- Best_Performer

---

## Dynamic Dataset Generation
The dataset is generated dynamically using a Python script. Every time the script is executed, new football match data is created with randomly generated values for:
- Goals Scored
- Goals Conceded
- Match Result (Win/Loss/Draw)
- Player Name
- Performance Level

The updated data is automatically saved as **football_data.csv**.

---

##  Dashboard Features
- Total Matches
- Total Wins
- Total Losses
- Total Draws
- Goal Difference
- Average Goals Scored
- Goals Scored vs Goals Conceded
- Match Result Distribution
- Player Performance Analysis
- Interactive Filters (Player & Result)

---

## Dynamic Dashboard Updates
The Power BI report is connected to the generated CSV dataset. Whenever the Python script creates a new dataset, the CSV file is updated. After refreshing the Power BI report, all KPI cards, charts, tables, and filters automatically update to display the latest football match statistics. This demonstrates seamless integration between Python and Power BI.
